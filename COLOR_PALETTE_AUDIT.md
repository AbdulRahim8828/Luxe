## ğŸ“Š Icon Color Rules

### **Rule 1: White Icons on Colored Backgrounds**
When icon is inside a colored container (gradient or solid):
```tsx
<div className="bg-amber-600">
  <Icon className="text-white" />
</div>
```

### **Rule 2: Colored Icons on White/Light Backgrounds**
When icon is on white or light background:
```tsx
<div className="bg-white">
  <Icon className="text-amber-600" />
</div>
```

### **Rule 3: Colored Icons on Matching Light Backgrounds**
When icon is on its own light background:
```tsx
<div className="bg-green-100">
  <Icon className="text-green-600" />
</div>
```

### **Rule 4: Dark Background Icons**
On dark backgrounds (footer, dark sections):
```tsx
<div className="bg-gray-900">
  <Icon className="text-amber-400" />
</div>
```
